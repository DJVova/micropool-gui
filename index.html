Micropool Setup

  1. Run the latest Swap Daemon and Sync. Latest Swap Daemon
  2. Enter a standard wallet address. (mining to exchanges is not supported)
  3. Change the daemon IP address if you are running the daemon on another machine.
  4. Close the Micropool and reopen to save settings.

Miner Setup

  1. Update the IP address to the machine running the Micropool.
  2. Set the port to 14650.
  3. Add worker names for multiple rigs.

Troubleshooting

Deamon connection

The error "connect ECONNREFUSED x.x.x.x:19950" is indicate that the daemon is likely not synced to the network; restart the micropool when the deamon is fully synchronized.
